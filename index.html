Learning Apache Spark

Ming Chen & Wenqiang Feng

Introduction

This repository contains mainly notes from learning Apache Spark by Ming Chen & Wenqiang Feng. We try to use the detailed demo code and examples to show how to use pyspark for big data mining. If you find your work wasnt cited in this note, please feel free to let us know.
